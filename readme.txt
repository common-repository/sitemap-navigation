== Description ==
Sitemap Navigation plugin can create a sitemap for your WordPress powered site. This is not another xml sitemap plugin, but rather a nice post sitemap or page sitemap navigation.

Some of its features include: support for multi-level categories and pages, category/page exclusion, multiple-page generation with navigation, permalink support, archive, RSS feed, xml sitemap, and much more.
